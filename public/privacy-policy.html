Privacy Policy

Clockify Contract Hours Tracker

Summary: This addon is developed for private use among friends and stores your contract and time tracking data locally in your browser. We do not collect, store, or transmit your personal data to any third-party servers. Access is restricted to invited users only.

1. Information We Collect

1.1 Data You Provide

  • Contract Information: Contract names, weekly hour commitments, start/end dates, vacation days
  • Notification Preferences: Your chosen settings for alerts and notifications
  • Clockify API Credentials: API key, workspace ID, and user ID (if you choose to enable API integration)

1.2 Data We Access from Clockify

When you enable Clockify API integration, we access:

  • Your time entries from Clockify
  • Project and workspace information
  • User profile data (name, email) for display purposes only
  • Tags and project categories

1.3 Private Use Scope

This addon is designed for private use among a small group of friends. Data access is limited to:

  • Invited users who have been granted access
  • Local browser storage on your personal device
  • Optional Clockify API integration (your choice)

2. How We Use Your Information

  • Calculate and display contract hour progress
  • Provide overtime analysis and trend tracking
  • Send notifications about contract milestones
  • Generate reports and visualizations
  • Sync data with your Clockify account (when enabled)

2.1 Friend Group Privacy

For this private addon:

  • Each user's data remains private and local to their browser
  • No data sharing between users unless explicitly exported by the user
  • The developer does not access or monitor individual user data
  • Access is restricted to invited friends only

3. Data Storage and Security

3.1 Local Storage

All your data is stored locally in your browser using:

  • Browser Local Storage: For settings and preferences
  • IndexedDB: For time entries and contract data
  • Clockify Settings: Stored within the Clockify addon system

3.2 No External Data Transmission

Important: We do not send your data to any external servers, analytics services, or third-party applications. All processing happens locally in your browser.

3.3 Private Addon Security

For this private addon among friends:

  • Access is restricted to invited users only
  • No personal information about users is collected by the developer
  • Each user maintains full control over their own data
  • The addon operates independently for each user

3.4 Clockify API Security

When using Clockify integration:

  • We use official Clockify API endpoints with secure HTTPS connections
  • Your API key is stored securely in your browser or Clockify's addon settings
  • We only request necessary permissions for the addon functionality

4. Data Sharing and Disclosure

We do not share, sell, rent, or disclose your personal information to third parties. The only data exchange occurs between:

  • Your browser and the addon interface
  • Your browser and Clockify's official API (when enabled)

5. Your Rights and Controls

5.1 Data Access

You can access all your data through:

  • The addon's export functionality
  • Browser developer tools (for technical users)
  • Clockify's own data export features

5.2 Data Deletion

You can delete your data by:

  • Using the addon's reset/clear data functionality
  • Clearing your browser's local storage
  • Uninstalling the addon from Clockify

5.3 API Integration Control

You have full control over Clockify API integration:

  • Enable or disable API access at any time
  • Revoke API permissions through Clockify settings
  • Use the addon offline without any API connection

6. Cookies and Tracking

This addon does not use:

  • Tracking cookies
  • Analytics services (Google Analytics, etc.)
  • Social media tracking pixels
  • Third-party advertising networks

7. Children's Privacy

This addon is designed for professional time tracking and is not intended for children under 13. We do not knowingly collect personal information from children.

8. International Data Transfers

Since all data processing occurs locally in your browser, there are no international data transfers by our addon. Any data transfers occur only between your browser and Clockify's servers (based on your Clockify account location).

9. Changes to This Privacy Policy

We may update this privacy policy to reflect changes in our practices or for legal reasons. We will notify users of any material changes through:

  • Addon update notifications
  • Updates to this privacy policy page
  • Notices within the Clockify addon interface
